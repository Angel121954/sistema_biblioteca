=== PhpSpreadsheet Ejemplo Estático ===

1️⃣ Abre una terminal en esta carpeta:
   cd C:\xampp\htdocs\phpSpreadsheet_export

2️⃣ Instala las dependencias:
   composer install

3️⃣ Inicia Apache desde XAMPP

4️⃣ Abre en tu navegador:
   http://localhost/phpSpreadsheet_export/exportar_excel.php

✅ Se descargará un archivo Excel llamado "usuarios_ejemplo.xlsx".
